# README

#### サイト名：FOR JOBER

#### 製品概要: サブスクリプションモデルの人材マッチングサイト
* 人材紹介の総合プラットフォーム
* 企業と求職者が自由にコンタクト可能

Ruby 2.6.5

AWS ALB
